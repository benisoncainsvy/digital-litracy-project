📌 Project Overview

This project is part of the Digital Literacy course, where I have taken on the role of a Student Digital Ambassador. The aim is to build awareness and practical understanding of essential digital skills among students.

The project covers five key areas:

Digital awareness
Online professional presence
Coding & collaboration tools
Email etiquette
Cybercrime awareness

All tasks are organized in this repository along with supporting files, screenshots, and reflections.
